This README txt file is for the project5 directory.

To compile this project:

You must first be in the project5 directory and then type:

    $ mvn compile

The resulting files will be in the target subdirectories:

   ./target/classes         -- the source code class files

To run the JavaFX project5 type:

   $ mvn exec:java -Dexec.mainClass=edu.uga.cs1302.quiz.GeographyQuiz
